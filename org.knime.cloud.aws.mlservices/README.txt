This plug-in provides functionality to use AWS Web Services.

This plug-in makes use of the following libraries/projects:

1. AWS Java SDK For Amazon Personalize, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
2. AWS Java SDK For Amazon Personalize Runtime, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
3. AWS Java SDK, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
4. AWS Java SDK For AWS IAM, https://aws.amazon.com/sdk-for-java/, Amazon Web Services, Inc.
2. Apache Avro, https://avro.apache.org, Apache Software Foundation

See the lib directory for more information.
